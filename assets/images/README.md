This directory should contain all image files used by the Safari Luxe website.

## Image Guidelines

- **Format:** JPG or PNG
- **Style:** Black & white or grayscale preferred
- **Content:** Wildlife, safari scenes, natural landscapes
- **Quality:** High resolution for web use
- **License:** Ensure proper licensing for all images

## Image Sources

Recommended free stock photo sources:
- Unsplash.com (search: safari, wildlife, elephant, leopard)
- Pexels.com (free stock photos)
- Pixabay.com (free images)

## Note

The website will function without images, but they are essential for the complete visual experience. Placeholder images or alt text will display if images are missing.
